# SSHConfigCheck
Proiect ITBI, decembrie 2025
Studenti: Amarie Elena-Carina, Oancea Stefania-Valentina

Proiectul este un script shell care primeste drept argument un fisier de configurare si contine diferite functionalitati: verificarea existentei fisierului si a permisiunilor, verificarea setarilor de securitate recomandate, detectarea optiunilor duplicate sau suprascrise.
