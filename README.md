# SSHConfigCheck
Proiect ITBI, decembrie 2025
Studenti: Amarie Elena-Carina, Oancea Stefania-Valentina

Proiectul consta intr-un script shell care primeste drept argument un fisier de configurare si are diferite functionalitati: verificarea existentei fisierului si a permisiunilor, verificarea setarilor de securitate recomandate(Port, PermitRootLogin, PasswordAuthentication, PermitEmptyPasswords, MaxAuthTries), detectarea optiunilor duplicate sau suprascrise.
